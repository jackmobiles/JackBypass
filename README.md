# JackBypass

JackBypass is a KernelSU module designed to optimize the Android environment for a smoother gaming experience in Free Fire by applying system-level tweaks automatically during boot.

## Features

* 🔒 **SELinux Lock**
  Automatically enforces SELinux to **Enforcing** mode by checking and restoring it every 60 seconds.

* 🌐 **Network Refresh**
  Resets selected networking rules and firewall configurations to maintain a clean network environment.

* 🧹 **Background Cache Cleaner**
  Periodically removes temporary game cache and unnecessary files to help keep storage clean.

* 🛡️ **System Environment Management**
  Applies system configuration adjustments to maintain a consistent runtime environment while the module is active.

* ⚙️ **Automatic Boot Service**
  Starts all module functions automatically after every device reboot.

## Requirements

* KernelSU
* Root access
* Android device compatible with KernelSU

## Installation

1. Open the KernelSU app.
2. Go to **Modules**.
3. Install `JackBypasss-V4.zip`.
4. Reboot your device.
